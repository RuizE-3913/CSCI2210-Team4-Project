UPON START THE SYSTEM LOADS:
5 Airports (PHX, LAX, LAS, DFW, ORD)
3 Aircraft with generated seats
3 Flights with different routes and schedules
3 Staff members assigned to flights
1 pre-existing confirmed booking

This ensures the system is immediately usable without manual setup.



===MENU OPTIONS===
1. Flight Management
2. Book a Flight
3. View Bookings
4. Airport Management
5. Payment (Card Only)
6. Staff Management
7. System Report
0. Exit



===BOOKING PROCESS===

- Select a flight from Flight Management
- Enter passenger details
- Choose an available seat
	- Booking is created (status = PENDING)
- Proceed to Payment Menu
- Enter card payment amount
	- If successful → Booking is CONFIRMED



===NOTES===

A flight must be selected before booking.
A booking must exist before payment can be processed.
Seats must be available before assignment.
